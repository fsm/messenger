# Messenger

Messenger is a Facebook Messenger target for [fsm](https://github.com/fsm/fsm).

## Environment Variables

When using this target, you must set two environment variables:

```
MESSENGER_VERIFY_TOKEN=""
MESSENGER_ACCESS_TOKEN=""
```
